
# PRODUCTION + MONEY ENGINE (PERSONAL LEASING SYSTEM)

## Features
- Microservices Architecture
- JWT Auth + Roles
- Event Driven System (RabbitMQ)
- Lead Generation + Outreach Engine
- Candidate + Job Matching
- Notification System
- Redis Cache Ready

## Start
docker-compose up -d

## Services Flow
Lead → Outreach → CRM → Job → Matching → Assignment → Notification
